## Overview

- Global, scalable, fully managed
- 100% uptime
- Public and private DNS zones
- Features:
	- Access managed via IAM
	- Integrated with Logging
	- DNS peering and forwarding
	- Anycast:
		- Multiple machines sharing IP
		- User routed to the closest machine
	- DNSSEC

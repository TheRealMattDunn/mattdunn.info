## Overview

- Relational database
- Supports:
	- SQL Server
	- MySQL
	- PostreSQL
- Auto replication, failover, backup, and point-in-time recovery (PITR)—built in HA
- Auto scales to 30 TB
- Scales manually to 96 cores / 624 GB RAM
- Read replicas available

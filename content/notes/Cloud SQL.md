## Overview

- Relational database
- Supports:
	- SQL Server
	- MySQL
	- PostreSQL
- Auto replication (within region), failover, backup, and point-in-time recovery (PITR)—built in HA
- Auto scales—30 TB max
- Vertcial scales manually to 96 cores / 624 GB RAM
- Read replicas available

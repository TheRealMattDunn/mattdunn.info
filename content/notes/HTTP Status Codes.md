## 100s - Information

- 100 Continue
- 101 Switching protocol

## 200s - Success

- 200 OK
- 201 Created
- 202 Accepted
- 204 Not content
- 206 Partial content

## 300s - Redirection

- 301 Moved permanently
- 304 Not modified
- 307 Temporary redirect
- 308 Permanent redirect

## 400s - Client Errors

- 400 Bad request
- 401 Unauthorized
- 403 Forbidden
- 404 Not found
- 408 Request timeout
- 429 Too many requests

## 500s - Server Errors

- 500 Internal server error
- 501 Not implemented
- 502 Bad gateway
- 503 Unavailable
- 504 Gateway timeout
- 511 Network authentication required
